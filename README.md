# OnlineCompiler

1) Fetch from repo:
  ```
  git clone https://github.com/madanimohadhabib/OnlineCompiler.git
  ```
2) Change to Onlinecompiler directory
  ```
  cd Onlinecompiler
  ```
3) Pipe requirements to Onlinecompiler folder:
  ```
  pip install -r requirements.txt
  ```
4) Run Django app:
  ```
  py manage.py runserver
  ```
